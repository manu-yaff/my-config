# Basic commands

- pwd

  - show current dir

- mkdir

  - create a folder

- rm

  - remove file

- rmdir

  - remove dir

- cd

  - navigate to folder

- touch

  - create file

- ls

  - list content

- clear

  - clear console

- _>_ (standard output)

  - redirect stdout

- _>>_ [file]

  - append output

- < (standard input)

  - redirect stdin. eg. wc < _file_

- -
  * match 0 or more chars
    - eg. ls \*.txt
- ;

  - execute multiple commands in the same line

- open
  - open file

- copy all files from one folder to another
  - cp
